# Dia 08 - Introdução ao Desenvolvimento Web com ASP.NET

## Tópicos do Dia

1. **Controllers e MVC** - Estrutura MVC, Actions, Model Binding
2. **Middleware** - Pipeline de requisições, custom middleware
3. **Dependency Injection** - Injeção de dependências no ASP.NET Core
4. **Razor Pages** - Páginas Razor, model binding
5. **Hosting e Deploy** - Configuração, ambientes, publicação

**Objetivo**: Compreender os fundamentos do desenvolvimento web com ASP.NET Core.
