# Exercícios — CI/CD e Automação

## Exercício 1: Pipeline Básico ⭐
Crie um pipeline GitHub Actions simples que:
- Faça build de uma aplicação .NET
- Execute testes unitários
- Publique artefatos

## Exercício 2: Deploy Automático ⭐⭐
Configure um pipeline que faça deploy para Azure App Service quando houver push na branch main.

## Exercício 3: Testes de Integração ⭐⭐⭐
Implemente um pipeline completo com:
- Build e testes unitários
- Testes de integração com banco de dados
- Análise de cobertura de código
- Deploy para staging
- Testes end-to-end
- Deploy para produção

## Exercício 4: Blue-Green Deployment ⭐⭐⭐⭐
Implemente estratégia de blue-green deployment usando slots do Azure App Service.

## Exercício 5: Monitoramento e Alertas ⭐⭐⭐⭐⭐
Configure monitoramento completo com Application Insights e alertas automáticos.

## Exercício 4: Desafio ⭐⭐⭐⭐
Crie solução otimizada e extensível.

## Exercício 5: Projeto Integrado ⭐⭐⭐⭐⭐
Combine conceitos em projeto realista.
