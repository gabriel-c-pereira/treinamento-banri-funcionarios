# Exercícios — Containerização

## Exercício 1: Dockerfile Básico ⭐
Crie um Dockerfile otimizado para uma aplicação ASP.NET Core que:
- Use multi-stage build
- Minimize o tamanho da imagem final
- Execute como usuário não-root

## Exercício 2: Docker Compose ⭐⭐
Configure um docker-compose.yml para uma aplicação completa com:
- API ASP.NET Core
- Banco PostgreSQL
- Redis para cache
- Volumes persistentes

## Exercício 3: Kubernetes Deployment ⭐⭐⭐
Crie manifests Kubernetes para deploy da aplicação incluindo:
- Deployment com réplicas
- Service para exposição
- ConfigMap para configurações
- Health checks (liveness e readiness)

## Exercício 4: Segurança em Containers ⭐⭐⭐⭐
Implemente boas práticas de segurança:
- Imagens sem vulnerabilidades
- Secrets management
- Network policies
- Resource limits

## Exercício 5: CI/CD com Containers ⭐⭐⭐⭐⭐
Configure pipeline completo que:
- Build de imagem Docker
- Scan de vulnerabilidades
- Push para registry
- Deploy no Kubernetes
- Rollback automático

## Exercício 4: Desafio ⭐⭐⭐⭐
Crie solução otimizada e extensível.

## Exercício 5: Projeto Integrado ⭐⭐⭐⭐⭐
Combine conceitos em projeto realista.
