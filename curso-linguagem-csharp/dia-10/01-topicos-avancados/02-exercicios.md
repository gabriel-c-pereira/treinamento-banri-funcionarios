# Exercícios — Tópicos Avançados

## Exercício 1: Generics Avançados ⭐

Crie uma classe genérica `Repository<T>` com constraints apropriadas:
- T deve ser uma classe com construtor padrão
- Implemente métodos CRUD básicos
- Adicione método genérico de busca por propriedade

## Exercício 2: Reflection ⭐⭐

Implemente um serviço de mapeamento usando reflection:
- Método para copiar propriedades entre objetos
- Suporte a mapeamento por nome de propriedade
- Tratamento de tipos diferentes (conversão automática)

## Exercício 3: Dynamic Objects ⭐⭐⭐

Crie um builder dinâmico para objetos de configuração:
- Use ExpandoObject para criar objetos dinâmicos
- Permita adicionar propriedades e métodos dinamicamente
- Implemente validação dinâmica de propriedades
