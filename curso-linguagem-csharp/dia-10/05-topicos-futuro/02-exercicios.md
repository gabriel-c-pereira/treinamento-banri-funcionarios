# Exercícios — Tópicos para o Futuro

## Exercício 1: Source Generator Básico ⭐
Crie um source generator que gere propriedades INotifyPropertyChanged automaticamente para classes marcadas com um atributo customizado.

## Exercício 2: Minimal API Completa ⭐⭐
Implemente uma API REST completa usando Minimal APIs com:
- CRUD operations
- Validação de entrada
- Middleware customizado
- Autenticação JWT

## Exercício 3: MAUI App Cross-Platform ⭐⭐⭐
Desenvolva uma aplicação MAUI que:
- Funcione em Android, iOS e Windows
- Use MVVM pattern
- Consuma uma API REST
- Persista dados localmente

## Exercício 4: Native AOT e Performance ⭐⭐⭐⭐
Otimize uma aplicação para Native AOT:
- Remova reflection incompatível
- Use tipos de valor apropriados
- Configure trimming correto
- Meça e compare performance

## Exercício 5: Cloud-Native com Azure ⭐⭐⭐⭐⭐
Implemente uma solução completa usando:
- Azure Functions
- Azure Container Apps
- Azure Cognitive Services
- Monitoramento e observabilidade

## Exercício 4: Desafio ⭐⭐⭐⭐
Crie solução otimizada e extensível.
